# Krowi's Brokers [Time]

A clean, simple DataBroker addon that displays the current time on your broker bar.

## Features

- **Time Display** - Show local time, server time, or both
- **Format Options** - Choose between 12-hour and 24-hour formats
- **Seconds Display** - Toggle seconds on/off (updates every second when enabled)
- **Colored Text** - Optional green coloring for the time display
- **Calendar Integration** - Left-click opens the in-game calendar
- **Time Manager** - Shift + Left-click opens the time manager
- **Clean Tooltip** - Simple tooltip showing both local and server time
- **Right-Click Menu** - Easy access to all settings

## Display Modes

- **Local Time** - Your computer's local time
- **Server Time** - The realm/server time
- **Both** - Display both times separated by a slash

## Usage

- **Left-Click**: Open Calendar
- **Shift + Left-Click**: Open Time Manager  
- **Right-Click**: Settings Menu
- **Hover**: Show tooltip with local and server time

## Installation

1. Download the addon
2. Extract to your `World of Warcraft\_retail_\Interface\AddOns\` folder
3. Restart WoW or reload UI (`/reload`)
4. The time will appear on any LibDataBroker display (like Bazooka or ChocolateBar)

## Requirements

- A LibDataBroker display addon (e.g., Bazooka, ChocolateBar, TitanPanel with LDB support)
- **Hide Unused**  Only show currencies you have.
- **Abbreviation**  None, 1k, or 1m format.
- **Header Visibility**  Show or hide specific currency groups.
- **Alphabetical Sorting**  Currencies sorted by name within each group.

### Configuration
- **Default Tooltip**  Choose which tooltip shows by default.
- **Realm Tracking**  Track all realms or current realm only.
- **Session Duration**  Select from 1, 2, 4, 8, 12, 24, or 48 hours.
- **Activity Check**  Configurable interval to extend active sessions.
- Right-click the broker button to access all settings quickly.
