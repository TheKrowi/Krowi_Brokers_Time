Krowi's Brokers [Time] is a clean, simple DataBroker addon that displays the current time on your broker bar. Choose between 12-hour and 24-hour formats, show local or server time, and access the calendar with a single click.

### Display Modes
Choose what appears on your data broker bar:
- **Local Time**  Your computer's local time.
- **Server Time**  The realm/server time.
- **Both**  Display both local and server time side by side.

### Time Format
Customize how time is displayed:
- **12-Hour Format**  Standard AM/PM display (e.g., 3:45 pm).
- **24-Hour Format**  Military time display (e.g., 15:45).
- **Show Seconds**  Toggle seconds display on or off.
- **Colored Text**  Optional green coloring for the time display.

### Tooltip
Simple and informative:
- Shows both local and server time.
- Clear usage hints for keyboard shortcuts.
- Clean, uncluttered design.

### Quick Actions
One-click access to time-related features:
- **Left-Click**  Opens the in-game calendar.
- **Shift + Left-Click**  Opens the Time Manager.
- **Right-Click**  Opens the settings menu.

### Configuration
- **Time Format**  Switch between 12-hour and 24-hour formats.
- **Display Mode**  Choose Local, Server, or Both.
- **Show Seconds**  Enable to update every second (default: every 30 seconds).
- **Colored Text**  Toggle green-colored time text.
- Right-click the broker button to access all settings quickly.

### Compatibility
Works with any DataBroker display addon:
- Bazooka
- ChocolateBar
- Titan Panel
- Fortress
- And more!
